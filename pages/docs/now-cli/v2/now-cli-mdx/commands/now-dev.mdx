## Dev

The `now dev` command is used to replicate the ZEIT Now deployment environment locally, allowing you to test your [Serverless Functions](/docs/v2/serverless-functions), without requiring you to deploy each time a change is made.

### Basic Usage

<Snippet dark text="now dev" />
<Caption>Using the <InlineCode>now dev</InlineCode> command from the root of a project directory.</Caption>

### Options Available

The following options can be passed when using the `now dev` command:

- `--build-env`
- `--debug`
- `--env`

For more information on options and their usage, refer to the [options section](#options).
