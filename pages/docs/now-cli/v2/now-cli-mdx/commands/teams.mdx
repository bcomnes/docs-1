## Teams

The `now teams` command is used to manage [teams](/docs/v2/advanced/platform/users-and-teams/) under an account, providing functionality to list, add, and invite new members to teams.

### Basic Usage

<Snippet dark text="now teams list" />
<Caption>Using the <InlineCode>now teams</InlineCode> command to list all teams under an account.</Caption>

### Extended Usage

<Snippet dark text="now teams add" />
<Caption>Using the <InlineCode>now teams</InlineCode> command to create a new team.</Caption>

<Snippet dark text="now teams invite [email]" />
<Caption>Using the <InlineCode>now teams</InlineCode> command to invite a new member to a team.</Caption>

### Options Available

The following options can be passed when using the `now teams` command:

- `--debug`
- `--global-config`
- `--help`
- `--local-config`

For more information on options and their usage, refer to the [options section](#options).
