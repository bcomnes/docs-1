## Whoami

The `now whoami` command is used to show the username of the user currently logged into [Now CLI](/download).

### Basic Usage

<Snippet dark text="now whoami" />
<Caption>Using the <InlineCode>now whoami</InlineCode> command to view the username of the user currently logged into Now CLI.</Caption>

### Global Options

The following [global options](#options) can be passed when using the `now whoami` command:

- `--debug`
- `--global-config`
- `--help`
- `--local-config`
- `--token`

For more information on global options and their usage, refer to the [options section](#options).
