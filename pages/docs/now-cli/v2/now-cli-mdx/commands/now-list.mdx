## List

The `now list` command, which can be shortened to `now ls`, is used to provide a list of your deployments along with information about them.

### Basic Usage

<Snippet dark text="now list" />
<Caption>Using the <InlineCode>now list</InlineCode> command to retrieve information about multiple deployments.</Caption>

### Extended Usage

<Snippet dark text="now list [project-name]" />
<Caption>Using the <InlineCode>now list</InlineCode> command to retrieve information about deployments for a specific project.</Caption>

<Snippet dark text="now list [project-name] [--meta foo=bar]" />
<Caption>Using the <InlineCode>now list</InlineCode> command to retrieve information about deployments filtered by metadata.</Caption>

### Options

The following options can be passed when using the `now inspect` command:

- `--all`
- `--debug`
- `--global-config`
- `--help`
- `--local-config`
- `--scope`
- `--token`

### Unique Options

These are options that only apply to the `now list` command, therefore, more information is provided.

- `--meta` - filters the project based on attached metadata

For more information on options and their usage, refer to the [options section](#options).
