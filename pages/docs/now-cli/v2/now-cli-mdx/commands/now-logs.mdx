## Logs

The `now logs` command is used to retrieve logs data for a specific deployment.

### Basic Usage

<Snippet dark text="now logs [deployment-url]" />
<Caption>Using the <InlineCode>now logs</InlineCode> command to retrieve logs for a specific deployment.</Caption>

### Options Available

The following options can be passed when using the `now logs` command:

- `--all`
- `--debug`
- `--global-config`
- `--help`
- `--local-config`
- `--scope`
- `--token`

### Unique Options

These are the options that only apply to this particular Now CLI [command](#commands), therefore, more information is provided.

- `--follow` - watch for additional data
- `--number` - the maximum number of logs to output
- `--output` - specify the output format
- `--since` - returns logs only after a specific date
- `--query` - returns logs against a search query
- `--until` - returns logs only before a specific date

For more information on options and their usage, refer to the [options section](#options).
