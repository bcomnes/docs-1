## Deploy

The `now deploy` command, commonly shortened to just `now`, is used to deploy projects and can be used from either the root of the project directory, or by providing a path to it.

### Basic Usage

<Snippet dark text="now" />
<Caption>Using the <InlineCode>now</InlineCode> command from the root of a project directory.</Caption>

### Extended Usage

<Snippet dark text="now [path-to-project]" />
<Caption>Using the <InlineCode>now</InlineCode> command and supplying a path to the root directory of the project.</Caption>

### Options Available

The following options can be passed when using the `now` command:

- `--build-env`
- `--debug`
- `--env`
- `--force`
- `--help`
- `--name`
- `--prod`
- `--public`
- `--regions`
- `--version`

For more information on options and their usage, refer to the [options section](#options).
